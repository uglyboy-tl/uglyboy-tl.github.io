### 无屏幕启动树莓派

#### 开启 WiFi
在未启动树莓派的状态下单独修改 /boot/wpa_supplicant.conf 文件配置 WiFi 的 SSID 和密码，这样树莓派启动后会自行读取 wpa_supplicant.conf 配置文件连接 WiFi 设备。

操作方法简单：将刷好 Raspbian 系统的 SD 卡用电脑读取。在 boot 分区，也就是树莓派的 /boot 目录下新建 wpa_supplicant.conf 文件，按照下面的参考格式填入内容并保存 wpa_supplicant.conf 文件。
```
country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
 
network={
ssid="网络的ssid"
psk="密码"
key_mgmt=WPA-PSK
priority=1
}
```
#### 开启 SSH
在 boot 分区新建一个文件，空白的即可，文件命名为 ssh。注意要小写且不要有任何扩展名。

树莓派在启动之后会在检测到这个文件之后自动启用 ssh 服务。随后即可通过登录路由器找到树莓派的 IP 地址，通过 ssh 连接到树莓派。

#### 树莓派默认用户名密码
- 用户名：pi
- 密码：raspberry

连接ssh时记得用如下命令登录：
``` bash
ssh pi@ip地址
```

### 树莓派基础优化
#### ssh 允许 root 用户登录
```bash
vi /etc/ssh/sshd_config
```
其中修改为：
```
# Authentication:
LoginGraceTime 120
# PermitRootLogin without-password
PermitRootLogin yes
StrictModes yes
```
此时就允许 root 登录 ssh 了。

注意生效需要重启 ssh 服务：
```bash
/etc/init.d/ssh restart
```

#### 修改用户名和组名
修改用户名需要用其他用户账号登录，一般可以用root登录后修改。

相关命令如下：
``` bash
# 修改用户名
usermod -l 新用户  旧用户 
# 修改家目录
usermod -d /home/新用户名 -m 新用户名
# 修改组名
groupmod -n 新用户名 旧用户名
# 修改 UID
usermod -u 1000 新用户名
# 查看用户 UID
id 新用户名
```

#### 修改树莓派的国内源
修改 /etc/apt/sources.list 文件：
```
deb https://mirrors.aliyun.com/raspbian/raspbian/ buster main contrib non-free rpi
```
修改 /etc/apt/sources.list.d/raspi.list 文件：
```
deb http://mirror.tuna.tsinghua.edu.cn/raspberrypi/ buster main
```

#### raspi-config 
执行树莓派专用配置程序进行优化：
```bash
raspi-config 
```
其中 **设置时区**、**设置语言编码**、**扩展磁盘空间** 和 **调整GPU所占内存** 是比较重要的调整内容。

### 一些个性化设置
#### oh_my_zsh
安装 oh_my_zsh 需要系统中已经安装了 git 和 zsh，然后执行如下命令:
```bash
sh -c "$(wget https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"
```
将 zsh 切换为默认终端的命令：
```bash
chsh -s /bin/zsh
```
在 .zshrc 中几个有用的小命令：
```bash
# 查看文件夹大小
alias size='f(){ du -h --max-depth=1 $1 | sort -hr; }; f'
# 查找文件
alias sek='f(){ find ./ -name $1; }; f'
# 查看自动启动的服务
alias auto='systemctl list-unit-files --type=service | grep enabled | more'
# 显示当前时间日期
alias now='date "+%Y-%m-%d %H:%M:%S"'
# 解压缩 tar 文件
alias untar='tar xvf'
# 查看本机的外网ip地址
alias ipe='curl https://www.ip.cn'
```
