新年第一天，重新起航

2022-01-01
心情
约 367 字 预计阅读 1 分钟

文章目录

  • 这里会写点什么?
  • 更新的频率会怎样?

不知不觉，2022年了，在新年的第一天，又重新鼓捣了一下 Github 的 Pages，重新搭起了个人主页。虽然以前的文章已经都不在了，不过，也算是一个全新的开始吧，新年第一天，重新起航

这里会写点什么?

还是简单一点，当作一个普通的 Blog 吧，也许会有点技术内容，也许会有些互联网大厂内幕，但还是以自己随手记的心情为主，不在乎谁来看，只当作留给未来的自己吧。

更新的频率会怎样?

iPad 上有一个软件叫 Metion 好像可以比较方便的与 GitHub 同步更新，那样或许更新的频率会高一点，不过我觉着怎么着也得一两周才有些更值得记录的内容。

毕竟，更加学术的文档以及还有一大堆的技术 wiki 的坑还等着我去填呢😂。

还有啥要说的吗?

太久不写文章，文字已经生疏了，也不知道还有没有可能找回当年的风采。

当然，现在的工作，可以天天宅在家里，终究是幸福的，没道理抱怨。

悠悠已经香香的睡了，今天的内容也就写这么多吧。
